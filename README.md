## 资源描述

RAW格式图片是相机直接拍摄出的一种格式，通常我们看到的图像是经过转换后的格式。在开发图像处理相关的产品时，特别是涉及到图像转码（如Bayer转RGB）的场景，输入图像通常需要是RAW格式的。
